# Retailer Sales Backend

## Features

- JWT Authentication
- Retailer CRUD
- CSV Retailer Import
- Bulk Retailer Assignment
- Redis caching
- Pagination + filtering
- Docker support

## Tech Stack

- Node.js
- Express
- Prisma
- PostgreSQL
- Redis

## Setup

```bash
npm install
```

```bash
npx prisma migrate dev
```

```bash
npm run seed
```

```bash
npm run dev
```

## Docker

```bash
docker compose up --build
```
